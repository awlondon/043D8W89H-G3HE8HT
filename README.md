# Rebar Cutting App

## Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install) (matches `pubspec.yaml` SDK constraint).
- A configured Flutter device or simulator:
  - Android emulator or connected device.
  - iOS simulator (macOS only).
  - Desktop or web target (if enabled in your Flutter setup).

## Setup
```bash
flutter pub get
```

## Run
```bash
flutter run
```

## Test
```bash
flutter test
```

## Lint
```bash
flutter analyze
```
